Вместо скучных фиксированых курсов валют я решил сделать проверку курса в режиме реального времени при помощи
API (условно) бесплатного ресурса https://www.currencyconverterapi.com/
Я вывел получения курса выбранной валюты (относительно рубля, рубль - дефолтная валюта в моем решении) 
в отдельную функцию класса CashCalculator - get_rate(). Ее описание есть в докстринге и строчных комментариях. 
Решение рабочее, брал подсказки из темы "Сетевые запросы" из вводной части курса.
